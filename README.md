Melhorias principais desta versao:

- tema visual principal alinhado ao mockup neon do produto

## Sobre

O **FF Boost** e uma ferramenta de otimizacao de desempenho para jogos em emuladores Android no Windows.

Desenvolvido para:

- Free Fire
- FPS mobile
- jogadores competitivos

---

## O Que Ele Faz

- Aumenta FPS
- Reduz uso de RAM
- Fecha apps inuteis automaticamente
- Prioriza o emulador
- Ativa modo desempenho do Windows
